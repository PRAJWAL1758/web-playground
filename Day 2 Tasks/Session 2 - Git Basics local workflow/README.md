# Web Playground
This project is created to practice Git basics and VS Code setup.

## Purpose
- Learn Git local workflow
- Practice commits and commit messages
- Understand basic project structure

## Git Commit Convention
This project follows the Conventional Commits style.

Format:
type: short description

Examples:
- init: initial project setup
- docs: update README
- chore: add gitignore

More information:
https://www.conventionalcommits.org/

## Tools Used
- Git
- Visual Studio Code
